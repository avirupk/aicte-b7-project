# 📰 TruthLens AI

TruthLens AI is an AI-powered fake news detection system built using Machine Learning and deployed with Streamlit.

---

## 🚀 Features

- TF-IDF + Logistic Regression Model
- Platt Scaling Calibration
- Conservative Fake News Threshold (0.75)
- Interactive Risk Dashboard
- Credibility & Fake Risk Gauges
- Analysis History with CSV Export
- OCR Image Text Extraction (Optional)
- Professional Newspaper UI

---

## 🧠 Model Details

- Feature Extraction: TF-IDF Vectorizer
- Classifier: Logistic Regression
- Calibration: Platt Scaling
- Threshold Strategy: Conservative to reduce false positives

---

## 🛠 Tech Stack

- Python
- Streamlit
- Scikit-learn
- Plotly
- Pandas
- Joblib
